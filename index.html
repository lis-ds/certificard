    AluraFlix - Catálogo de filmes

    Catálogo de filmes, com uma seleção inicial de filmes já apresentada. O usuário é convidado a montar seu próprio catálogo, podendo adicionar e deletar filmes, e ordená-los por ordem crescente ou decrescente do ano de lançamento.

    Funcionalidades:

  • Apresentação de uma galeria com os pôsteres e títulos dos filmes inclusos;
  • Campos para adição de novos filmes a partir da inserção do título, url da imagem do pôster e ano de lançamento;
  • Lista para seleção de filmes a serem deletados do catálogo;
  • Botões para indicar o ordenamento desejado.

    Tabela de classificação

    Tabela de classificação de campeonato. Use para acompanhar seu time favorito ou para jogar com os amigos.
    Para cada vitória são marcados 3 pontos, e para cada empate, 1 ponto.

    Funcionalidades:

  • Botões para adição e subtração de vitórias, derrotas e empates;
  • Calcula o total de pontos de cada participante;
  • Destaque para o participante com mais pontos (com critério de desempate);
  • Ordenação por classificação ou ordem alfabética;
  • Opção para adicionar e deletar participantes da lista;
  • Opção para zerar todas as pontuações ou limpar a tabela por completo.

    • GitHub / CodePen
